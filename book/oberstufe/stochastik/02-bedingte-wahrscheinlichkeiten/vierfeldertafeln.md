# Vierfeldertafeln

## Aufgabe 1

::::multievent
:::snippet{#aufgabe}
Vervollständige die folgende Vierfeldertafel.

|                | $U$      | $\overline{U}$ |          |
| -------------- | -------- | -------------- | -------- |
| $K$            | 0,4      | 0,3            | {z{0,7}} |
| $\overline{K}$ | 0,2      | 0,1            | {z{0,3}} |
|                | {z{0,6}} | {z{0,4}}       | 1        |

:::
::::

## Aufgabe 2

::::multievent
:::snippet{#aufgabe}
Berechne mit Hilfe der Vierfeldertafel aus Aufgabe 1 die bedingten Wahrscheinlichkeiten.

(1) $ P_K(U) $ = {z{0,57}}

(2) $ P_U(K) $ = {z{0,67}}
:::
::::

## Aufgabe 3

::::multievent
:::snippet{#aufgabe}
Berechne die Wahrscheinlichkeit, dass $\overline{U}$ eintritt, wenn $K$ eingetreten ist. Runde auf zwei Nachkommastellen.

Antwort: {z{0,43}}
:::

:::collapsible{title="Hilfe"}
Die bedingte Wahrscheinlichkeit $ P_{K}(\overline{U}) $ berechnet sich wie folgt:

$$ P_{K}(\overline{U}) = \frac{P(K \cap \overline{U})}{P(K)} $$

In der Vierfeldertafel findest du die Werte für $ P(K \cap \overline{U}) $ und $ P(K) $.
:::
