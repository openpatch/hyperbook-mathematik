# Mündliche Prüfung 02

1. Definieren Sie das Skalarprodukt zweier Vektoren. Erläutern Sie eine praktische Anwendung für das Skalarprodukt.

2. Die Punkte $A(-2|3|-3)$, $B(-1|1|2)$, $C(1|-2|3)$ und $D(0|0|-2)$ sind in dieser Reihenfolge die Eckpunkte eines Parallelogramms.

    a) Weisen Sie nach, dass es sich wirklich um ein Parallelogramm handelt.

    b) Berechnen Sie den Schnittwinkel der Diagonalen des Parallelogramms.

3. Bestimmen Sie einen Normalenvektor der Ebene, die durch die Punkte $A$, $B$ und $C$ aufgespannt wird.
